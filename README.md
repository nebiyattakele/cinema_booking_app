
# Cinema Booking App

A Flutter application that allows users to browse, book, and rate movies in a cinema.

## Description

This app is developed using **Flutter** and **Firebase**. It allows users to:

- Browse **Now Showing** and **Upcoming Movies**
- View detailed information about each movie
- **Book tickets** and **choose seats** (as seats are selected, the total cost updates)
- **Rate movies** after watching, with ratings used by others to decide which movie to watch

The app also includes **user profiles**, **favorite movies**, and a **review system** for better movie recommendations.

---

## Features

- Browse movies by category: Now Showing, Upcoming  
- Movie details page with description, rating, duration, and trailer  
- Seat selection with dynamic pricing  
- Book tickets and view booking confirmation  
- Rate movies after watching  
- Firebase integration for data storage and authentication
